# Autonomous Self-Healing Infrastructure & Security Patching Engine

An AI-powered incident response workflow built using FastAPI, n8n, Groq LLMs, and Gmail.

## Features

- Production crash detection
- AI root cause analysis
- Severity classification
- AI-generated remediation suggestions
- Security guardrails
- Human approval workflow
- Automated incident notifications

## Tech Stack

- Python
- FastAPI
- n8n
- Groq LLM
- Gmail Integration

## Workflow

Crash → Investigation → Repair → Guardrail → Human Approval
